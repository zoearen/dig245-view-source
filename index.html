Viewing the Source Code of Binary Piano on Google Chrome

  • Open a tab in Google Chrome.
  • Go to binarypiano.com .
  • Click the three dots on the upper right corner of the browser screen
  • Click "More Tools"
  • Click "Developer Tools." A panel should pop up on the right side of the screen.
  • Navigate to the "Sources" tab
  • Under "binarypiano.com," click (index)
  • Now you can see the source code for binarypiano.com!

Other Dev Tools Actions

  • Edit HTML in Elements: you can edit HTML through Dev Tools in the elements tab - changes will appear as if they have actually been made to the source code.
  • Clear Cookies: In the Application panel go to Storage > Cookies to either clear all cookies or an individual cookie.
  • Color Palette and Picker: In Styles, click a color swatch from the library, or use the color picker to select any color on the screen.
